1) Download or clone the repository in Xampp htdocs folder.
2) Start Apache and MySQL from the Xampp App.
3) Open http://localhost:80/phpmyadmin or http://localhost:8080/phpmyadmin.
4) Goto import and select nerdit.sql file and click on import.
5) Open http://localhost/nerdit/ and you are good to go.
